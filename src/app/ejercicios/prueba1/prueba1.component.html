          "La puedes usar como pista, antes de ver el código" 
          y posteriormente el código.
        
          El componente tiene una propiedad infoSuperheroes que es un objeto que contiene 
          información sobre varios superhéroes en forma de pares clave-valor. 
          Cada clave es el nombre del superhéroe y cada valor es un objeto que contiene 
          información como su nombre real, poderes y equipo.
  
          La clase también tiene dos propiedades adicionales: alias y info. 
          alias es una cadena vacía que se usará para buscar información sobre 
          un superhéroe específico. info es una propiedad que se usará para 
          almacenar la información del superhéroe una vez que se haya encontrado.
  
          El componente también tiene un método llamado mostrarInfoSuperHeroe(),
          que se ejecuta cuando se hace clic en un botón en el HTML del componente. 
          Este método busca en el objeto infoSuperheroes la información correspondiente 
          al superhéroe cuyo nombre se encuentra en la propiedad alias.
  
          Si el superhéroe existe, la información se asigna a la propiedad info. 
          Si no existe, se imprime un mensaje en la consola indicando que el superhéroe 
          no se encuentra en la base de datos.
  
          En resumen, este componente Angular muestra información sobre superhéroes 
          en una interfaz de usuario. Al hacer clic en un botón, el componente busca 
          información sobre un superhéroe específico y la muestra en la interfaz.

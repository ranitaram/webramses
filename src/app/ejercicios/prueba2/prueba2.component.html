        Enunciado Ejercicio 2:
        Dada una ruta absoluta de un archivo (de un sistema linux o basado en unix)
        crear una función que la simplifique

        Ejemplo:
        simplificarRuta("/home/");             // Salida: /home
        simplificarRuta("/x/./y/../../z/");    // Salida: /z
        simplificarRuta("/../");               // Salida: /
        simplificarRuta("/home//pruebas/");    // Salida: /home/pruebas

Simplificar ruta

ejemplo: /x/./y/../../z/

Ruta simplificada: {{ rutaSimplificada }}

            Para resolver este ejercicio en Angular y TypeScript, podríamos seguir los siguientes pasos:

            Crear una función en un componente o servicio de Angular que tome una ruta absoluta de un archivo como entrada.

            Dividir la ruta en sus componentes usando el carácter '/' como separador.

            Iterar sobre cada uno de los componentes de la ruta y manejar los siguientes casos:
            Si el componente actual es un directorio vacío ("") o un punto (".") entonces no hacemos nada y pasamos al siguiente componente.

            Si el componente actual es un doble punto (".."), eliminamos el componente anterior de nuestra pila de componentes.

            En cualquier otro caso, agregamos el componente actual a nuestra pila de componentes.
            Unimos los componentes de la pila de componentes para construir la ruta simplificada.
        
            tiene dos propiedades: ruta y rutaSimplificada. 
            ruta es una cadena de texto que representa la ruta que ingresará el usuario
            y rutaSimplificada es la cadena de texto resultante de simplificar la ruta ingresada.

            Después esta una función que se ejecutará cuando el usuario haga clic en el botón "Simplificar". 
            Lo que hace esta función es llamar a la función simplificarRuta pasando la ruta ingresada por el usuario como argumento 
            y asignar el resultado a la propiedad rutaSimplificada.

            Desues esta la función que realiza la simplificación de la ruta ingresada por el usuario. 
            Lo que hace es dividir la ruta en componentes utilizando el separador de directorios ("/") 
            y luego recorre cada componente de la ruta.

            Si el componente es una cadena vacía o un punto (".") no hace nada. 
            Si el componente es dos puntos (".."), se elimina el último componente de la pila. 
            Si el componente es cualquier otra cosa, se agrega a la pila.

            Finalmente, se crea una nueva cadena de texto que representa la ruta simplificada 
            a partir de los componentes que quedan en la pila y se retorna esta cadena.

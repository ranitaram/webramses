        Ejercicio 5:
        Dado un array de personas con nombres y apellidos
        crear una función para ordenar el array por los apellidos
        en orden alfabético
         
        Ejemplo:
        ordenarPorApellidos([
        "Juan Perez", 
        "Laura Guzman", 
        "Carlos Rivera", 
        "Natalia Hernandez", 
        "Jose Martinez"
        ]);

Nombre de las personas

  • Juan Perez
  • Laura Guzman
  • Carlos Rivera
  • Jose Martinez
  • Natalia Hernandez

Nombres ordenados alfabéticamente por apellidos

  • {{nombreOrdenado}}
          A Continuación la solución explicada 
          "La puedes usar como pista, antes de ver el código" 
          y posteriormente el código.
        
            Crea una función llamada ordenarPorApellidos que reciba como parámetro un array de personas 
            con nombres y apellidos.

            Dentro de la función, declara una variable llamada nombresOrdenados y asígnale el resultado de
            llamar al método sort() en el array de personas. 
            
            Pasaremos una función de comparación como argumento a sort().
            En la función de comparación, utiliza la función split(" ") en las variables a y b 
            para separar los nombres y apellidos por espacios. 
            Esto creará arreglos donde el primer elemento será el 
            nombre y el segundo elemento será el apellido.

            Luego, utiliza localeCompare() en los apellidos (a.split(" ")[1] y b.split(" ")[1]) 
            para realizar una comparación alfabética y determinar el orden de los apellidos.
            Finalmente, retorna el array nombresOrdenados al final de la función.

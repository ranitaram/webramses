        Ejercicio 3:
        Dadas dos fechas, crea un programa que proporcione la diferencia de dias
        entre ellas
 
        Ejemplo:
        diferenciaDeDias('Dec 1, 2023', 'Dec 24, 2023');  // Salida: 23

La diferencia de días es: {{ diferencia }}

          A Continuación la solución explicada 
          "La puedes usar como pista, antes de ver el código" 
          y posteriormente el código.
        
            fecha1: string = '';: Declaración de la variable fecha1 con tipo de dato string e inicialización con un valor vacío.
            fecha2: string = '';: Declaración de la variable fecha2 con tipo de dato string e inicialización con un valor vacío.
            diferencia: number = 0;: Declaración de la variable diferencia con tipo de dato number e inicialización con un valor de 0.

            calcularDiferencia(): Declaración de la función calcularDiferencia que se ejecutará cuando el usuario haga clic en el botón.
            const date1 = new Date(this.fecha1);: Crea un objeto de fecha (Date) a partir del valor de fecha1, 
            que es una cadena de caracteres que representa una fecha.
            
            const date2 = new Date(this.fecha2);: Crea un objeto de fecha (Date) a partir del valor de fecha2, 
            que es una cadena de caracteres que representa una fecha.
            
            const diffTime = Math.abs(date2.getTime() - date1.getTime());: Obtiene la diferencia de tiempo (en milisegundos) 
            entre date1 y date2, utilizando el método getTime() 
            
            para obtener el valor en milisegundos de cada objeto de fecha. La función Math.abs() 
            se utiliza para asegurarse de que el resultado 
            sea siempre un número positivo.
            const diffDays = Math.floor(diffTime / (1000 * 60 * 60 * 24));: Convierte la diferencia de tiempo en días, dividiendo el resultado 
            por el número de milisegundos en un día (1000 * 60 * 60 * 24) y redondeando hacia abajo utilizando la función Math.floor().
            this.diferencia = diffDays;: Asigna el valor de diffDays a la variable diferencia, para mostrar el resultado al usuario.
        
            En resumen, la función calcularDiferencia() toma los valores de fecha1 y fecha2, los convierte en objetos de fecha, 
            calcula la diferencia de tiempo en días y asigna el resultado a la variable diferencia, que se muestra al usuario.

        Ejercicio 6:
        Dado un año actual, crea un programa que me muestre 
        los años bisiestos que habrá en los proximos 30 años
         
        Ejemplo:
        bisiestos(2023);
         
        Salida:
        2024
        2028
        2032
        2036
        2040
        2044
        2048
        2052

Años bisiestos

Año actual: {{ anosLimite }}

Años bisiestos en los próximos 30 años:

  • {{ ano }}
          A Continuación la solución explicada 
          "La puedes usar como pista, antes de ver el código" 
          y posteriormente el código.
        
            La clase tiene una propiedad anosLimite que es de tipo number. 
            Esta propiedad se utiliza para almacenar el año actual obtenido mediante 
            new Date().getFullYear() en el constructor de la clase.

            El constructor de la clase se ejecuta cuando se crea una instancia y se 
            encarga de asignar el valor del año actual a la propiedad anosLimite.

            La clase también tiene un método llamado getProximosAnosBisiestos que acepta dos parámetros: 
            comenzarAno y contador. Este método devuelve un array de números (number[]) que representa los 
            años bisiestos en los próximos contador años a partir del año comenzarAno.

            Dentro de getProximosAnosBisiestos, se declara una variable local anosBisiestos que es 
            inicializada como un array vacío para almacenar los años bisiestos.

            Luego se declara una variable local ano que se inicializa con el valor de comenzarAno + 1. 
            Esta variable se utiliza para iterar a través de los años desde el año siguiente al año de inicio.

            A continuación, hay un bucle while que se ejecuta mientras la longitud de anosBisiestos sea 
            menor que contador. Esto asegura que se encuentren los próximos contador años bisiestos.

            Dentro del bucle, se verifica si el año actual (ano) es un año bisiesto llamando al
            método esAnoBisiesto pasando el valor de ano. Si es bisiesto, 
            se agrega al array anosBisiestos utilizando el método push.

            Después de eso, se incrementa el valor de ano en 1 para pasar al 
            siguiente año y continuar el bucle.

            Finalmente, se devuelve el array anosBisiestos que contiene los años bisiestos encontrados.

            La clase también tiene un método llamado esAnoBisiesto que acepta un parámetro ano de tipo number y 
            devuelve un valor booleano (boolean). Este método verifica si un año es bisiesto utilizando una 
            expresión condicional que evalúa si el año es divisible por 4 y no divisible por 100,

            o si es divisible por 400. Si la condición se cumple, devuelve true, de lo contrario, devuelve false.

            En resumen, este código define una clase Prueba6Component que calcula y almacena los 
            años bisiestos en los próximos contador años a partir de un año de inicio dado.

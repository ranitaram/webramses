        Ejercicio 7:
       
        Crea un programa que cuente las palabras de un texto,
        el usuario debe de poder ingresar el texto en un campo y 
        el programa debe de contar cuantas palabras existen en ese texto.

Palabras contadas: {{ contador }}

          A Continuación la solución explicada 
          "La puedes usar como pista, antes de ver el código" 
          y posteriormente el código.
        
            se definen dos variables en la clase AppComponent. 
            texto es una variable de tipo string que almacenará el texto ingresado por el usuario,
            y contador es una variable de tipo number que almacenará la cantidad de palabras
            en el texto.

            contarPalabras() es una función que se ejecuta cuando se hace clic en el botón "Contar palabras". 
            En esta función, se verifica si el texto ingresado por el usuario, 
            después de eliminar los espacios en blanco al inicio y al final con trim(), está vacío. 
            Si el texto está vacío, se establece contador en 0, indicando que no hay palabras. 

            En caso contrario, se divide el texto en palabras utilizando el método split(' '), 
            que divide el texto en cada espacio en blanco y devuelve un array de palabras. 
            Luego, se asigna la longitud de ese array a contador, 
            indicando la cantidad de palabras en el texto.

            La lógic    a de este código es simple: si no hay texto, el contador es 0; de lo contrario, 
            se cuentan las palabras en el texto y se actualiza el contador.
